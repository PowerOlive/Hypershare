How does it work?

Simply upload some files to Hypershare and share the download URL with your friends. Files are automaticly removed after 7 days.

Is there a upload limit?

No, Maybe. We don't enforce a file size limit. We don't set limits on the actual file upload, there could however be a hardware limit which prevents huge file uploads.
